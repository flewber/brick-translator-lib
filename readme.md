# Brick Translator Lib

## Prerequisits

- install rebrick: `pip3 install rebrick`
- download and extract parts.csv from https://rebrickable.com/downloads/
- put parts.csv in top level directory of this repo
- create and log in to your Rebrickable account
- on https://rebrickable.com/ click on account --> settings --> API
- copy your API key (generate one first if you don't already have one)
- paste your key into the sample `keys_example.py` file
- rename the sample file to `keys.py`

## Generating the Part Translations Table
run `python3 get-translations.py`
